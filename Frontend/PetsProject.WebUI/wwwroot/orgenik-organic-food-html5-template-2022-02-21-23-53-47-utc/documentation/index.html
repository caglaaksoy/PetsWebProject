A) Introduction - top

Orgenik is a beautiful one-page template specialized in landing page and also a responsive HTML5 Bootstrap template. It is a great choice for those who are in the reading business. Orgenik is specially built to showcase and sell your books or eBooks. As a changing trend of commerce that adapts to e-books more and more needs the adequate publicity.

Orgenik offers book reviews, brief about authors, contact form and about section. It is a template thought for visual aesthetics and a well-presented image. Orgenik is feature-rich and carefully designed to help you quickly create a landing page for your books. It allows authors and web retailers to promote quality content.

Template Features

  • Latest Bootstrap 4.3.2
  • Multipage and landing template
  • Sticky Navigation
  • W3x validate Validated
  • Responsive Template
  • FontAwesome icons
  • Themeify Icons
  • Google Web Fonts
  • Working Ajax Contact form
  • Documentation
  • Easy to customize and user friendly
