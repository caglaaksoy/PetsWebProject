Getting Started

This template is divided in two folders:

Folder Description
/build This is the compiled version. Includes regular CSS and ready to be uploaded to your server.
/source Use this folder if you're used to work with Sass.

Installation

Upload the content inside the build folder to your server.

Classes

There are some core classes that defines the layout and design. They are global and not bound to any specific module.

Alignment Description
.center Add this class to center content.
.center-desktop Center content starting from the desktop media query.
.center-tablet Center content starting from the tablet media query.
.center-mobile Center content starting from the mobile media query.
.middle Align content vertically when height is unknown.
Backgrounds Description
.bg-dark Solid background where "dark" can be replaced with "light", "primary", etc.
.bg-gradient-blue Gradient background where "blue" can be replaced with "cyan", "green", etc.
.bg-image-01 Image-based backgrounds where "01" can be replaced with "02", "03", etc.
.overlay Adds an overlay with opacity on top of backgrounds.
.overlay-shape-01 Adds an overlay shape on top of backgrounds where "01" can be replaced with "02", "03", etc.
Sizes Description
.full-screen Adding this class will make sure the module goes full screen.
.full-width This class will make sure the element goes full width.
.full-width-tablet Useful if you'd like an element to go full width on tablets and mobile devices.
.full-width-mobile Useful if you'd like an element to go full width only on mobile.
.max-width-s Set a small maximum width of an element.
.max-width-m Set a medium maximum width of an element.
.max-width-l Set a large maximum width of an element.
.min-width-s Set a small minimum width of an element.
.min-width-m Set a medium minimum width of an element.
.min-width-l Set a large minimum width of an element.
Opacity Description
.opacity-s Adds a small amount of opacity.
.opacity-m Adds a medium amount of opacity.
.opacity-l Adds a large amount of opacity.
Rounded Description
.rounded Adds a border radius to an element.
.rounded-bottom Adds a border radius to the bottom of an element.
.rounded-top Adds a border radius to the top of an element.
Shadow Description
.shadow-s Adds a small shadow to an element.
.shadow-m Adds a medium shadow to an element.
.shadow-l Adds a large shadow to an element.
.shadow-none Removes an existing shadow from an element.
Spacing Description
.padding This is the main inner spacing of a module.
.padding-bottom This is the main bottom spacing of a module.
.padding-top This is the main top spacing of a module.
.padding-top-tablet Add spacing above content starting from the tablet media query.
.margin-bottom Add spacing below an element inside a module.
.margin-top Add spacing above an element inside a module.
.space Add default spacing around an element.
.space-bottom Add default spacing to the bottom of an element. Note that "bottom" can be replaced with "top", "right" and "left".
.space-none Remove any spacing that is around an element.

Grid

The simple grid below is based on 5 columns. You can add "no-gutter" as a class to remove spacing between columns.

Example: <div class="row">
  <div class="col-one-third"></div>
  <div class="col-one-third"></div>
  <div class="col-one-third"></div>
</div>

You can also have unequal columns.

Example: <div class="row">
  <div class="col-two-thirds"></div>
  <div class="col-one-third"></div>
</div>

Add class .reverse-order to reverse the order of columns on tablets and mobiles.

Example: <div class="row">
  <div class="col-one-half"></div>
  <div class="col-one-half"></div>
</div>
<div class="row reverse-order">
  <div class="col-one-half"></div>
  <div class="col-one-half"></div>
</div>

The illustration below shows above code.

Add class .min-two-columns for a minimum of two columns on mobile.

Example: <div class="row min-two-columns">
  <div class="col-one-fourth"></div>
  <div class="col-one-fourth"></div>
  <div class="col-one-fourth"></div>
  <div class="col-one-fourth"></div>
</div>

Icons

The easy to use material icons by Google are excellent for web projects. Based on the 8-point grid, they are preferably displayed at a multiple of 24px.

Example: <i class="material-icons">face</i>

Typography

Listed below are all default styles for headings. Headings <h1> through <h6>, are available.

Example: <h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Lead

The main purpose of the lead style is to support headings as a subtitle.

Example: <p class="lead"></p>

Paragraph

With an increased font-size and line-height, the paragraph style is great for readability.

Example: <p class="paragraph"></p>

Lists

Below is the code for unordered lists.

Example: <ul>
 <li>List item one.</li>
 <li>List item two.</li>
 <li>List item three.</li>
</ul>

Below is the code for ordered lists.

Example: <ol>
 <li>List item one.</li>
 <li>List item two.</li>
 <li>List item three.</li>
</ol>

You can also add class "inline" to make horizontal lists.

Example: <ul class="inline">
 <li><a href="#">List item one.</a></li>
 <li><a href="#">List item two.</a></li>
 <li><a href="#">List item three.</a></li>
</ul>

Buttons

Buttons are divided into types: primary, secondary and other states such as disabled.

Example: <a href="#" class="button button-primary" role="button">Primary</a>
<a href="#" class="button button-secondary" role="button">Secondary</a>
<a href="#" class="button button-disabled" role="button">Disabled</a>

To change the size, you can add class "button-l" for a large button. There are four pre-defined sizes: large, default, medium and small.

Forms

Input fields are divided into types and states. Below code is a default input:

Example: <form>
 <div class="form-group">
  <label for="name">Your Name:</label>
  <input id="name" type="text" name="name">
 </div>
</form>

Dropdown:

Example: <form>
 <div class="form-group">
  <label for="country">Country:</label>
  <div class="select-wrapper">
   <select id="country" name="country">
    <option selected="" disabled="" value="">Choose country…</option>
   </select>
  </div>
 </div>
</form>

Input area:

Example: <form>
 <div class="form-group">
  <label for="email">Message:</label>
  <textarea id="message" name="message"></textarea>
  <span class="form-help">500 characters max.</span>
 </div>
</form>

Switch:

Example: <form>
 <div class="form-group">
  <label>Switch:</label>
  <input id="s1" class="switch" type="checkbox" checked>
  <label for="s1" class="switch"></label>
 </div>
</form>

Checkboxes:

Example: <form>
 <div class="form-group">
  <label>Checkboxes:</label>
  <input id="c1" type="checkbox" name="item-one">
  <label for="c1" class="checkbox">Item one.</label>
  <input id="c2" type="checkbox" name="item-two">
  <label for="c2" class="checkbox">Item two.</label>
 </div>
</form>

Radio buttons:

Example: <form>
 <div class="form-group">
  <label>Radio Buttons:</label>
  <input id="r1" type="radio" name="item" checked>
  <label for="r1" class="radio">Item one.</label>
  <input id="r2" type="radio" name="item">
  <label for="r2" class="radio">Item two.</label>
 </div>
</form>

Alerts

Use alerts to give users quick feedback on performed tasks.

Example: <div class="alert alert-success" role="alert">Great! This is awesome.</div>
Type Description
.alert-default Neutral and default notification.
.alert-error Displays an error message.
.alert-success Displays a success message.
.alert-warning Displays a warning.

Cards

A card is a great content container.

Example: <div class="card card-content">

Dark Mode

If you want to turn a module dark, add class .dark to the section.

Example: <section class="bg-dark dark"></section>
